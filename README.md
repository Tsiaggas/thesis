# Greek Text Sentiment Analysis

A deep learning system for sentiment analysis of Greek text, classifying content into 3 categories (Negative, Positive, Neutral) using BERT models.

## Live Demo

Try the live demo on Hugging Face Spaces:
**[https://huggingface.co/spaces/tsiaggas/GreekText-sentiment-analysis](https://huggingface.co/spaces/tsiaggas/GreekText-sentiment-analysis)**

## Description

This project aims to develop an effective sentiment analysis system for Greek texts, focusing on product reviews and general content. It utilizes state-of-the-art Natural Language Processing (NLP) techniques, specifically fine-tuning a pre-trained BERT model for Greek texts.

## Model Access

The trained model is available on the Hugging Face Hub:
**[https://huggingface.co/tsiaggas/fine-tuned-for-sentiment-3class](https://huggingface.co/tsiaggas/fine-tuned-for-sentiment-3class)**

It can be used directly with the following Python code:

```python
from transformers import AutoModelForSequenceClassification, AutoTokenizer

# Load model and tokenizer
model = AutoModelForSequenceClassification.from_pretrained("tsiaggas/fine-tuned-for-sentiment-3class")
tokenizer = AutoTokenizer.from_pretrained("tsiaggas/fine-tuned-for-sentiment-3class")
```

## Installation and Execution

### Prerequisites

- Python 3.8+
- CUDA (optional for GPU acceleration)

### Installation

1. Clone the repository:

```bash
git clone https://github.com/your-username/greek-sentiment-analysis.git
cd greek-sentiment-analysis
```

2. Create and activate a Python virtual environment:

```bash
# Create environment
python -m venv .venv

# Activate on Windows
.\.venv\Scripts\activate

# Activate on Linux/Mac
source .venv/bin/activate
```

3. Install the required libraries:

```bash
pip install -r requirements.txt
```

### Running the Web Application

The web application can use either the local model (if available) or the model from the Hugging Face Hub:

```bash
cd web_app
python app.py
```

The configuration is set in the `USE_HUGGINGFACE_HUB` parameter in the `web_app/app.py` file.

### Running Visualizations

To create advanced metric visualizations:

```bash
python utils/advanced_metrics.py
```

## Key Features

- **BERT Model Fine-tuning**: Adapting BERT for 3-class sentiment classification
- **Dataset Processing and Combination**: Combining data to create a balanced 3-class dataset
- **Evaluation and Visualization**: Calculation and visualization of advanced evaluation metrics
- **Demonstration Web Application**: Interactive application to showcase the model's functionality
- **Hugging Face Hub Upload**: Ability to share the model via the Hugging Face Hub

## Results

The model achieves high accuracy in sentiment classification of Greek texts:

- **Accuracy**: ~97.3%
- **F1 Score (Weighted)**: ~97.3%
- **Precision (Weighted)**: ~97.3%
- **Recall (Weighted)**: ~97.3%

Detailed results and visualizations are generated by the `advanced_metrics.py` script.
